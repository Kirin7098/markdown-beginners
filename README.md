**语法总结**
```
Markdown是一种轻量级的标记语言
标题：使用 # 表示， # 越多标题级别越低，如 # 一级标题 、 ## 二级标题 。
无序列表： - 或 * 开头，如 - 列表项1 
有序列表：数字加 . 开头，如 1. 列表项1 。
加粗：用 ** 包裹文本，如 **加粗文本** 。
斜体：用 * 包裹文本，如 *斜体文本* 。
链接：格式为 [链接文本](链接地址) ，如 [百度](https://www.baidu.com) 。
图片：格式为 ![替代文本](图片地址) 。![[]]
待办：- [ ]  - [x]
代码块：```
分隔线: ---
制表：|-|-|-|
数学公式：$公式$  $$公式居中$$
